# solveMaze

## 소개
C 언어를 활용하여 스택 기반 DFS로 미로 탈출 가능 여부를 판단하는 프로그램
## 파일 구성
- 'maze.h': Point, Stack 구조체 정의 및 함수 선언
- 'maze.c': 스택 연산(push, pop, top)과 DFS 탐색 로직 구현
- 'main.c': 시작/ 도착 좌표 설정, 미로찾기 결과 출력
## 동작 방식
1. 시작 좌표를 스택에 push 한다.
2. 스택의 top 좌표가 도착좌표와 같다면 성공을 리턴한다.
3. 상/하/좌/우 순서로 이동 가능한 칸(벽이 아니고 방문하지 않은 칸)을 찾아 하나만 push 한다.
4. 이동할 곳이 없다면 스택에서 pop을 하여 이전의 위치로 되돌아간다.
5. 스택이 빌 때까지 도착하지 못하면 실패를 반환한다.
## 빌드 및 실행

```bash
make
./maze
```
## 스택 구현 이유
미로 찾기 과정에서 경로가 없을 때 직전의 위치로 돌아가야하기 떄문에 최근 위치를 불러오는 것에 최적화되어있는 스택을 활용하였다.
## DFS를 사용한 이유
이 문제는 미로를 "탈출할 수 있는지의 여부"만 판단하면 되고, 최단 경로를 구할 필요가 없다. DFS는 한 방향으로 갈 수 있는 만큼 탐색하다가 막히면 pop으로 되돌아오는 방식으로, 경로 존재 여부를 확인하는 데 적절하다.
단, 최단 경로가 필요하다면 BFS를 사용하는 것이 더 적합하다.
